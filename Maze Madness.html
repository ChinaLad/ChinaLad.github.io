Maze Generation

Kruskal's algorithm

The idea of this method of generation is to use the walls of the maze. We start with a maze with walls placed everywhere. We place every cell in its own set of cells. We randomly pick a wall, remove it and union the set of cells on the left and the right of the wall. A wall is removed only when the sets on the left and the right are disjoint.

Prim's algorithm

Recursive algorithm

We start with no walls in the maze. We split it into 4 segments by placing one vertical and one horizontal wall from across the whole maze. We make 3 holes in the wall (there can be more than 1 wall connecting two of the segments). We recursivelly divide the grid as much as we can and make these holes like we didi for the first split.

Wilson's algorithm

We set a random cell as a part of the maze. We pick a random cell and do a random walk until we reach a cell that is part of the maze. Then we make the path into a coridor of the maze and all the cells along the path become part of the maze. Then we continue picking random cells and doing random walks until there are no cells left.
